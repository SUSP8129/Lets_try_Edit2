1. What Is "Edit2"?

"Edit2" is a browser-based text editor created by RasPython3.

The developer, RasPython3, told me that the idea was to create a simple text editor for Chromebook development because there were no easy-to-use text editors available.

After that, he continued to make improvements, and the current version was created.

This text editor is still being improved, and further evolution is expected.

For more information about RasPython3, please check "4. Additional Information" at the bottom of this page.

3. Is "Edit2" Easy to Use?

Edit2 is useful because it can be used in combination with the browser's spell checker.

It is also simple and easy to use, being one-file, lightweight, and excellent in that it has a replace function.

In addition, it can be used in place of a notepad.

It is also multifunctional, as explained in the video above, with support for DeepL translator and easy to adjust indentation.

At this point, I personally think it is better to use it for daily use rather than for serious development.

Why don't you all give it a try?
